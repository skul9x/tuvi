## [0.1.0] - 2026-01-30

### Added
- **Mystical UI**: Complete visual overhaul with "Mystical Luxury" theme.
    - Implemented split layout Landing Page.
    - Added "Void Purple" & "Divine Gold" color palette.
    - Added glassmorphism components (Form Card, Buttons).
    - Integrated `Cinzel` and `Inter` fonts.
    - Added CSS animations (Pulse, Shimmer, Spin).
- **Privacy Policy**: Added `/chinh-sach-bao-mat` page.
- **SEO**: Added `robots.ts` and `sitemap.ts`.

### Changed
- Refactored `globals.css` to use Tailwind CSS v4 variables.
- Updated `CalendarInput` component to match new theme.
- Updated `layout.tsx` to include new fonts and base styles.

### Fixed
- Resolved build issues related to Tailwind v4 and Turbopack configuration.
